General evaluation

70% project
30% final test


Project details

You need to form groups of 6-8 people to create a reco solution, chatbot solution, or reco and chatbot solution.



The code needs to be in a GitHub repo, you can either create an organization or use one already created.



The project needs to have:

Data prep
Machine learning algorithm(s)
Deployment
MLOps with tests


There needs to be a test process that executes every time there is a pull request. This is similar to what you would be doing in a real job.



You can use this repo as a template for your project: https://github.com/miguelgfierro/project_template



Also, the project needs to solve a business process with AI, and you can consider taking different ML roles available in AI:

Product Manager (PM)
Data Engineer
Data Scientist
Machine Learning Engineer
MLOps Engineer
Please note that the role should be chosen based on what role you would like to have in the future, and that it is ok to repeat roles in the project.



One important detail, please consider this project as something useful to you. Some examples:

If you are thinking of doing a startup, use this project as a feature for your product.
If you are building a project portfolio on GitHub, add this project to your portfolio.
If you are thinking of doing a final year project in reco or chatbots, use this project as the first chapter of your work.
If you are passionate about a specific area, build something that helps you (i.e. if you are into football, create a recommender that gets personalized news of your favorite team).


Evaluation method for the project

50% of the grade is the group contribution, and 50% is the individual contribution.
Each student needs to submit 2 PDFs (the personal report and the group report).
The evaluation method is listed as follows:

For the project:

Did they create a recommender/chatbot? (5/10)
Did they do an EDA? (1/10)
Did they do a deployment? (1/10)
Did they have MLOps? (1/10)
Are they following good development practices (work on branches, add code via PRs, comment on issues, libraries that are called by notebooks, etc)? (1/10)
Does Miguel see evidence of exceptional ability as a group? (1/10)
       NOTE: Make sure that points 1-5 are clear in the group report. Point 6 is only evaluated by Miguel.

For individual contributions:

Individual contributions. (2/10)
What have you done on top of the work of others. (2/10)
What have you done to help others. (2/10)
What you could have done better to improve. (2/10)
Does Miguel see evidence of exceptional ability? (2/10)
       NOTE: Make sure that points 1-4 are clear in the group report. Point 5 is only evaluated by Miguel.
